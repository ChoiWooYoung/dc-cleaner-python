# dc-cleaner-python
파이썬 Selenium을 이용한 DCINSIDE 게시글 및 댓글 클리너

## 환경
* Windows 10
* Python 3.8.x

## 필요한 것들
* Python 3.8.x
* Selenium

## 사용방법
1. run.py를 다운받은 후, user-agent 및 Webdriver PATH 변경
2. cmd나 Python IDLE 에서 프로그램 실행

## 주의할 점
* Headless 모드라 웹브라우저가 보이지 않음
* 갤러리 번호를 잘못 입력할 시 프로그램이 종료됨
* CAPTCHA는 직접 사이트에 들어가서 풀어야함
